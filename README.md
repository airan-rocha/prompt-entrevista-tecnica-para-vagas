# Prompt Entrevista Técnica Para Vagas

O objetivo desse repostório é disponibilizar os arquivos de uma atividade em um dos cursos da plataforma DIO

Nesse repositório há dois arquivos da atividade:

* [PROMPT - entrevista tecnica](https://github.com/airan-rocha/prompt-entrevista-tecnica-para-vagas/blob/main/PROMPT%20-%20entrevista%20tecnica) - esse arquivo contém o prompt que deverá ser inserido na IA
* [PROMPT - respota da entrevista tecnica](https://github.com/airan-rocha/prompt-entrevista-tecnica-para-vagas/blob/main/PROMPT%20-%20respota%20da%20entrevista%20tecnica) - esse arquivo contém as respostas e a minha interação com a IA após informar o prompt acima.
